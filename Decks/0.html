What is a datatype?

"A particular kind of data item, as defined by the values it can take, the programming language used, or the operations that can be performed on it."

What is a datatype?

Data Type Description Example
String Single words or sentences surrounded by double or single quotes
 		 "puppies", "lots of puppies!"
NUMBER Whole numbers or decimals 42, 1024, 3.14159
Boolean Represents either true or false true, false
​Array Collections of Data ['Superman', 'Batman', 'Spider-Man']
Object Many things! We'll get into this later { oooo so mysterious }

Strings, booleans, numbers, and objects are all primitives. Arrays are derived from object.

Words

  • Variables: store data types in a computer’s memory so that they can be referenced later
  • Keywords: reserved words in a language that cannot be used for variable or function names
  • There are two different variable keywords used, named let and const. However, at the end of the day, they're still going to be a variable.
  • let implies mutability, while const implies a constant

Arrays:

Collections of Data

Arrays are great for:

  • Storing data

  • Enumerating data (i.e., using an index to find them)

  • Arrays compose a data structure that is similar in concept to a list

  • Elements

  • In JavaScript, arrays can dynamically grow and shrink in size (different than some other languages)

Arrays, Continued

const friends = ['Moe', 'Larry', 'Curly'];
=> ['Moe', 'Larry', 'Curly']

// First friend
const firstFriend = friends[0];
	=> 'Moe'
// Get the last friend
const lastFriend = friends[2]
=> 'Curly'

Array literal:

Array Methods

  • a.toString() - Returns a string with the toString() of each element separated by commas.

  • a.pop() - Removes and returns the last item.

  • a.push(item1, ..., itemN) - Adds one or more items to the end.

  • a.reverse() - Reverses the array.

  • a.shift() - Removes and returns the first item.

  • a.unshift([item]) - Prepends items to the start of the array.

But wait, there's more!

  • Array.forEach()
  • Array.every()
  • Array.some()
